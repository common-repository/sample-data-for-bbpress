== Description ==
Sample data for bbpress

Use this plugin to create dummy data for users, forum, topics, replies to see how bbPress works or to test your own plugin or theme with bbPress.
Run this for unit testing, performance checks, ...

Starting point: Tools > bbPress SampleData

Credits to:

* Stephen Edgar for his structure and data in https://bbpress.trac.wordpress.org/ticket/2516 
* Daniel Dvorkin for his ideas and approach in bbpFauxData (https://github.com/MZAWeb/bbpFauxData)
* Rob Mehew for his view on adding dummy data on https://www.epicwebs.co.uk/bbpress-tutorials/test-data-for-bbpress-plugin/

== Installation ==
Option 1:

1. On your dashboard, go to Plugins > Add new
1. Search for *Sample data for bbPress*
1. Install and activate the plugin
1. Run bbPress SampleData from Tools

Option 2:

1. Unzip the contents to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Run bbPress SampleData from Tools
